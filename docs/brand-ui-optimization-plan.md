# Phase 48: Korean Luxury VAVAW Brand/UI Optimization Plan

## Objective
Upgrade the visual identity of the VAVAW ecosystem into a premium Korean-inspired luxury brand system before custom domain launch. This involves purely UI/UX enhancements without altering backend logic, database schema, or production API routes.

## Core Visual Principles
- **Vibe:** Korean luxury, cinematic, minimal, editorial, trustworthy, high-end but not flashy.
- **Avoid:** Generic SaaS look, cheap gradients, crowded layouts, random bright colors.
- **Elements:** Generous spacing, strong typography hierarchy, clean editorial composition, soft glass/subtle borders.
- **App-specific Palettes:**
  - `main`: black / ivory / silver
  - `main/cosmetic`: ivory / beige / clean gold
  - `beauty`: ivory / blush / espresso
  - `franchise`: black / ivory / champagne amber
  - `admin`: slate / white / blue

## Execution Tasks
### 1. apps/main
- **Hero:** Refine composition to feel cinematic and spacious.
- **Navigation/Header:** Refine VAVAW identity.
- **Business Cards:** Apply premium, editorial treatment.
- **/cosmetic:** Enhance product-first trust, refine hierarchy.
- **/contact:** Ensure styling consistency with luxury vibe.

### 2. apps/beauty
- **Hero:** Soft Korean spa/editorial mood.
- **Sections:** Improve service/trust areas.
- **Forms:** Polish lead forms for elegance and accessibility.

### 3. apps/franchise
- **Hero:** Premium business opportunity vibe.
- **Trust Sections:** Strengthen value proposition hierarchy.
- **Tone:** Confident, premium, investment-ready (black/ivory/champagne/amber).

### 4. apps/admin
- **Layout:** Improve sidebar, dashboard cards, page headers.
- **Tables:** Enhance readability and spacing (Leads, Audit Logs).
- **Forms/Badges:** Standardize spacing and status badges.
- **Tone:** Clean internal productivity dashboard (slate/white/blue).

### 5. Shared Consistency & Accessibility
- Standardize border-radius, card shadows, section spacing, typography scale, input focus states across the ecosystem.
- Ensure all forms have labels, buttons have visible focus states, and sufficient color contrast is maintained.
