# Launch Media Asset Checklist

This checklist defines the required media assets for the VAVAW Main homepage and Cosmetic landing page. Upload these via the Admin dashboard (`/media`) before the custom domain launch.

## 1. Main Homepage Assets (Site Key: `main`)

| ID | Purpose | Recommended Size | Format | Max Size |
| :--- | :--- | :--- | :--- | :--- |
| **`main-hero-atmosphere`** | Cinematic homepage hero background or atmosphere visual. | 2400x1600 or larger | WEBP/JPG | 1.5MB |
| **`main-cosmetic-preview`** | Preview card / ecosystem card for VAVAW Cosmetic. | 1600x1200 | WEBP/JPG | 1MB |
| **`main-beauty-preview`** | Preview card / ecosystem card for VAVAW Beauty & Co. | 1600x1200 | WEBP/JPG | 1MB |
| **`main-franchise-preview`** | Preview card / ecosystem card for VAVAW Franchise. | 1600x1200 | WEBP/JPG | 1MB |

## 2. Cosmetic Page Assets (Site Key: `cosmetic`)

| :--- | :--- | :--- | :--- | :--- |

## 3. Upload Guidelines & Naming Conventions

### Upload Location
- Navigate to **Admin Dashboard -> Media**.
- Select the appropriate `site_key` (`main` or `cosmetic`).
- Set type to `image`.

### Naming Conventions
Follow the exact IDs listed above when saving files locally before uploading to maintain consistency (e.g., `main-hero-atmosphere.webp`).

### Alt Text Guidelines
Alt text should be descriptive but not keyword-stuffed. It must describe the image for accessibility and SEO.
- **Good**: "VAVAW cinematic beauty atmosphere", "VAVAW Cosmetic clean beauty editorial", "Soft cosmetic texture and ritual detail"
- **Bad**: "makeup", "cosmetics buy online", "image123"

### What NOT to Upload
- Do not upload uncompressed RAW, TIFF, or BMP files.
- Do not upload random, low-quality stock images.
- Do not upload private customer images or images with unclear licensing.
- Do not upload files exceeding 5MB (the system will reject them).
