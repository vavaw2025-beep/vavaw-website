# Supabase Database

This folder contains the database schema, migrations, and seed files for the VAVAW CMS.

## Directory Structure
- `migrations/` - SQL files applied sequentially to the database schema.
- `seed/` - Initial data to insert into the database.

## Phase 15 Notice
Currently, we are in Phase 15. The schema has been planned and defined, but the application is not yet connected to a live Supabase instance.
